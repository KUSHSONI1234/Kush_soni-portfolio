    Here are a few projects I've worked on recently.

    Quiz Website

    I have created a fully responsive quiz app using HTML, CSS, and JavaScript, complete with great animations. In this app, users first log in and then are taken to the home page. Before starting the quiz, there is a guide for users to follow. Each quiz has a specific time limit, and after completing the quiz, users can log out of their accounts.

    Chat Application

    I developed a chat application using HTML, CSS, JavaScript, jQuery, and AJAX. jQuery manages user interactions, and AJAX allows messages to be sent to the server without reloading the page, providing a smoother user experience.

    E-commerce Website

    I created a fully responsive e-commerce website using HTML and CSS, focusing on learning CSS techniques. This project features user login, product sections like Packages, Services, Gallery, Reviews, and Contact, and a cart section for managing selected items. I incorporated CSS media queries, grid, flexbox, animations, positioning, typography, and z-index to enhance the site's design and functionality.

    Garden Website

    During my internship at Adept Digitals Private Limited, I developed a garden website that is fully mobile responsive. In the contact form, I added JavaScript functionalities for validation. Additionally, when the user clicks the "Submit Now" button, all submitted data is displayed on the same page.

    Tea-Station Website

    I developed and maintained Full Stack responsive websites using HTML, CSS, and JavaScript, along with modern frameworks like React and Angular. I created interactive animations and transitions to enhance user engagement and improve the visual appeal of the websites. I conducted cross-browser testing and debugging to ensure compatibility and optimize performance. Additionally, I implemented form validations and error handling to improve data integrity and the user experience in web forms.

    Quiz Website

    This project displays the Air and Noise quality levels around you and provides you solutions as well as cures to those problems. Surveyed the vicinity of GSFC University, Vadodara and developed an innovative IoT based solution to monitor the Air quality as well as Sound quality levels remotely. Results were authenticated by the GSFC Ltd, Safety and Health Department. Plotted a variety of real time graphs and histograms to visualize the data and provided solutions for the problems accordingly in the mobile application.
